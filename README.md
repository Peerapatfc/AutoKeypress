# AutoKeypress

A Python script that automatically presses the Enter key every 1 second for eFootball on Windows.

## Features
- Focus-steal mode: Works while you use other applications
- F10 to stop the script

## Requirements
- Python 3.x
- Windows OS

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python auto_keypress.py
```

Press **F10** to stop the script at any time.
